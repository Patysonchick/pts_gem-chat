# pts_gem-chat

Unlimited and uncensored chat with Google Gemini models.

## Configuring `.env`

### Create `.env` file with next content, for example

`API_KEY=...
MODEL=gemini-2.0-flash-thinking-exp-01-21
API_VERSION=v1beta
PROXY=socks5://127.0.0.1:2080`

If you won't use proxy, leave the field blank.
